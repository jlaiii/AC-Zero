Core Eradication Modules

Anti-Cheat System Termination

Targets and removes all lingering trace files, logs, and services from major anti-cheat platforms, including Easy Anti-Cheat (EAC), BattlEye, and Riot Vanguard components.

Game Launcher Data Annihilation

Thoroughly cleans residual configuration files, caches, and user data left by platforms such as Steam and Epic Games, ensuring a completely clean environment for application launches.

Specific Game Trace Erasure

Deep cleaning protocols targeting specific game directories and user profile data. This module focuses on eliminating traces related to titles like Rust and Fortnite.

Active Process Termination

Safely terminates all associated running processes before initiating cleaning (e.g., Steam, RustClient, Epic Games services). This ensures that critical files are unlocked and fully removable.

Windows OS Optimization

Executes essential operating system cleanup, including the complete clearance of the Recycle Bin, removal of Windows prefetch files, and reduction of residual OS clutter.

Registry Trace Elimination

Includes dedicated functionality to scan, locate, and remove specific, known registry entries and system traces associated with game components and anti-cheat systems to ensure maximum integrity.

Operational Protocol

1

Initiation & Validation

The script verifies execution environment compatibility (Windows) and automatically requests elevated administrator privileges, which are mandatory for system-level file access.

2

Process Lockdown

All target processes (Game launchers, Anti-cheats, Clients) are forcefully terminated to prevent file usage conflicts and enable complete data removal.

3

Systematic Execution

File paths (Appdata, ProgramData) and the Registry are systematically scanned, logged, and cleared for all persistent and temporary traces as defined by the protocol.
